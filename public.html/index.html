Linux, Git, and Docker Commands


Linux Commands

  • pwd Print Working Directory - it will print the path starting from the root to the working directory
  • ls List - it will print all files and directories/folders in the current directory as a list
  • cd Change Directory - it will change the current working directory (but only a subdirectory)
  • mkdir Make Directory - will create a directory within the current working directory
  • mv Move Files/Directory - it will "move" the files/directories by copying the original file and pasting it into a new directory and deleting the original one
  • touch Creates New File - it will create a new file in the current working directory; can also update timestamp of file
  • vi Text Editor - used to edit files

Git Commands

  • git init - creates a git repository in the current directory
  • git add - adds the file to the commit - UNKNOWN
  • git commit - modifications within the tree - UNKNOWN
